# Mappin

A web-based tool for visualizing glacier ice flow velocity data from NASA's [ITS_LIVE](https://its-live.jpl.nasa.gov/) (Inter-mission Time Series of Land Ice Velocity and Elevation) project.

## Features

- **Interactive map and time series charting** for exploring ice velocity across decades of data
- **Shareable URLs** that encode the full application state, providing a live velocity plot that can be referenced or embedded in papers
- **CSV export** for offline analysis
